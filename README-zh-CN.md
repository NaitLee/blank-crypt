# 空白加密 blank-crypt
*用以将文本加密成空白字符（空格，制表符，换行符）的算法*

## 介绍
许多其他算法都将文本加密成不可读的字符串，这样会给骇客招来注意。

但是，此算法将文本加密成不可见的内容。

这样，即使放在公开环境里，您的秘密将仍然是秘密。

## 使用
- 安装 *python3*
- Clone 此仓库。
- 将您的秘密写在一个文件里。
- 用 Python 3 打开 `prompt.py`，输入您的文件名和密码，将创建一份经加密的文件。
- 您可以打开此文件，看一片空白。
- 使用同样的方式您可以解密此文件，只要您知道密码。
- 更高级的使用详见 `blankcryptLib.py`。
  - 您可以这样将此算法包括在您的作品中：
  - `from blankcryptLib import crypt`
  - 现在您可以使用其中的 `crypt()` 函数。
